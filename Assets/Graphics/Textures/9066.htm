Texture Pack Description: 

This pack contains 3 1024x1024 metal textures with normal maps (alpha as height) and .xcfs for each.

These textures were created for UFO:AI to replace non GPL versions.

Edit: Now includes tileable versions of each texture by request, and the tileable versions have been significantly optimised for better performance. 

Edit 2: Now includes a fourth texture with an alternate version, normals maps for each and all tileable.

 

Edit 3: There are now three zips:

metals-tiny.zip contains just the tileable textures

metals-nm.zip contains just the normal maps

metals.zip contains everything, tileable textures, orignal non tileable textures, normal maps and xcfs.
